## 1. プロジェクト概要

### 1.1 目的

オリコン顧客満足度調査のプレスリリース作成業務を自動化し、**手動1-2時間 → 自動5分**に短縮する。

### 1.2 要件

- **完全オフライン動作** (機密情報保護のためAPI接続禁止)
- **GUI操作**: 元データExcelを選択するだけで完成Wordを出力
- **処理内容**:
  1. 元データExcel → リリース用表Excelへデータ転記
  2. 表をPNG画像化
  3. Wordテンプレートへ反映・画像差し替え
  4. 完成版Word出力

### 1.3 技術スタック

- **言語**: Excel VBA (Microsoft Office 2016以降)
- **参照設定**:
  - Microsoft Excel Object Library
  - Microsoft Word Object Library

### 1.4 開発体制

- **司令官**: Claude Sonnet 4.5
- **実装AI**: Claude Sonnet 4.5, GPT-4o
- **QA AI**: Gemini 2.5 Flash
- **調査AI**: Perplexity Sonar Pro (必要時)

## 8. 技術メモ

### 8.1 VBA抽出スクリプトの改善点

**現在の抽出パターン**:
```python
vba_pattern = r'```(?:vba|vb)\n(.*?)```'
```

**問題点**:
- 複数のコードブロックがある場合、最長のもののみ保存
- Attribute行の検出が不安定

**改善案**:
```python
# Attribute VB_Nameを必須とする
vba_pattern = r'```(?:vba|vb)\n(Attribute VB_Name.*?)```'

# または、見出し(###)とコードブロックをセットで抽出
section_pattern = r'### (Module\d_\w+)\.bas\n\n```(?:vba|vb)\n(.*?)```'
```
