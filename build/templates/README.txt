============================================================
  Release Creator - ポータブル版
  ユーザーガイド
============================================================

■ はじめに
  Release Creator は、オリコン顧客満足度ランキングの
  TOPICS・プレスリリースを自動生成するツールです。
  このポータブル版は Python のインストール不要で動作します。


■ 初回セットアップ（1回だけ）

  1. 共有フォルダの「ReleaseCreator.bat」をダブルクリック
  2. 自動的にローカル（お使いのPC）にコピーされます
  3. デスクトップに「Release Creator」ショートカットが作成されます
  4. セットアップ完了後、自動的にアプリが起動します

  ※ 初回は数分かかる場合があります


■ 通常の起動方法

  デスクトップの「Release Creator」ショートカットをダブルクリック
  → ブラウザが自動的に開きます


■ 終了方法

  方法1: ランチャーのコンソールウィンドウを閉じる
  方法2: stop.bat を実行する


■ アップデート

  共有フォルダのバージョンが更新された場合、
  次回起動時に自動的に差分更新されます。
  特別な操作は不要です。


■ トラブルシューティング

  【アプリが起動しない場合】
  1. start-debug.bat を実行してエラーメッセージを確認
  2. ウイルス対策ソフトが python.exe をブロックしていないか確認
     → IT部門にホワイトリスト登録を依頼してください

  【ブラウザが開かない場合】
  1. 手動で http://localhost:8501 をブラウザで開く
  2. ポートが使用中の場合は http://localhost:8502 ~ 8510 を試す

  【「ポートが見つかりません」エラー】
  1. stop.bat を実行して既存プロセスを停止
  2. 再度 ReleaseCreator.bat を実行

  【画面が正しく表示されない場合】
  1. ブラウザのキャッシュをクリア（Ctrl + Shift + Delete）
  2. 別のブラウザ（Chrome, Edge）で試す

  【上記で解決しない場合】
  IT部門またはシステム管理者にお問い合わせください。


■ ファイル構成（参考）

  release-creator-portable/
  ├── ReleaseCreator.bat    ← メインランチャー（これだけ覚えればOK）
  ├── start-debug.bat       ← トラブル時のデバッグ用
  ├── stop.bat              ← アプリ停止用
  ├── VERSION.txt           ← バージョン情報
  ├── README.txt            ← このファイル
  ├── python/               ← Python実行環境（変更不要）
  └── app/                  ← アプリケーション本体（変更不要）
